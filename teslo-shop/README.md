# Teslo API

1. Clonar proyecto
2. Ejecutar

```
yarn install
```

4. Clonar el archivo `.env.template` y renombrarlo como `.env`
5. Cambiar las variables de entorno
6. Levantar la base de datos

```
docer-compose up -d
```
